Detailed Description

Proxy class to call methods on a Characteristic such as ReadValue, WriteValue etc.

Note
It is different than the above Characteristic class as that class 'implements' a characteristic and so it is implemented on this system, and is should be used by only the peripheral. While this is a proxy to an already existing characteristic provided by some other remote device
